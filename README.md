
# Trabajo Práctico Integrador - Programación 1

## Descripción del proyecto
Este proyecto consiste en el desarrollo de una aplicación en Python para gestionar y analizar información relacionada con países. Se aplican conceptos fundamentales de Programación 1, como manejo de listas, diccionarios, funciones, estructuras condicionales y repetitivas, ordenamientos y cálculo de estadísticas.

## Estructura del proyecto
El proyecto está organizado en los siguientes archivos y carpetas:
- main.py: Archivo principal para la ejecución del programa.
- funciones.py: Contiene las funciones utilizadas para el procesamiento de datos.
- datos/: Carpeta que contiene archivos con información de países.
- README.md: Documentación del proyecto.




## Instrucciones de ejecución
